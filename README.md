# Dumb Simple Linkedin Auto Endorser

## 1. Installation

Run in console:

```bash
npm i
```

## 2. Setup

### Add people

Open `kudos.txt` and add people Linkedin urls one by line.
 
**They must be in your connections!**

See the file for first line as an example.

### Add credentials

Open `.env` and enter your Linkedin username and password, like so:

```
EMAIL=you_email@example.com
PASSWORD=Y0urPa$$word
```

**Disclaimer** - code is dumb simple, you username and password is not stored or passed anywhere.

## 3. Run

Run in console:

```bash
npm start
```
